# go-todo
Using Go and React to build a ToDo Application

### Replace MongoDB URI
replace the connectionstring in middleware > middleware.go in order to perform connection to ur Mongodb Database

### Installation
client
```sh
$ cd client
$ npm install 
$ npm start
```

server
```sh
$ cd server
$ go mod init github.com/<username>/<directory> 
$ go install
$ go run main.go
```
